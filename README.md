# Odysseus — Second Brain

This is the shared knowledge vault between Aloy and Odysseus.

## Structure

- `notes/` — General notes and learnings
- `memory/` — Key facts, decisions, context Odysseus should retain
- `resources/` — Articles, transcripts, reference material

- Aloy can audit all changes via git history
