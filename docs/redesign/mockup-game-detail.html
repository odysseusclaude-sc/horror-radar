Design Mockup Notes — Game Detail (Autopsy Page)

  • Section navigation: Sticky sidebar on desktop with jump links (Overview, Timeline, Creator Impact, Events, Community). Collapses to sticky horizontal pill bar on mobile.
  • Overview: Game header with cover placeholder, key stats grid, badges for subgenre, demo availability, developer track record, and price tier.
  • Timeline: CSS+SVG chart with labeled phase bands (Demo, Launch, Discovery, Settling). Zoom controls (7D/30D/All) and visible "Drag to select range" hint (D-11).
  • Phase cards: Non-hovering cards with explicit "Click to zoom to this period" signifier, addressing the audit finding about hover-lift without click affordance.
  • Creator Impact: Card layout instead of table. Each card shows avatar placeholder, name, subscriber count, video title, horizontal impact bar with score, and date. Much clearer hierarchy than a flat table.
  • Events: Grouped by type (YouTube, Reddit, Steam Updates) with expandable sections. Each group shows event count. Event items use colored dots for type differentiation alongside text labels.
  • Community: Summary cards for Reddit mentions, Steam discussions, Twitch streams, and achievement completion.
  • Breadcrumb: "Database > Hollowfield" navigation (CM-04).
  • Accessibility: Skip link, semantic HTML (section, article, nav), aria-labels, role attributes, focus-visible outlines, corrected color values.
  • Responsive: Sidebar collapses to horizontal pill bar on mobile. Stats grid adapts from 3-col to 2-col to 1-col.
