Navigation Restructuring

The current 4-page structure serves data exploration well but misses the discovery, comparison, and persistence workflows that all four personas need. The proposed navigation adds targeted entry points without abandoning the existing structure.

Current Navigation (4 pages)

📊
Database
Game table with filters, sorting, pagination
🔥
Radar Pick
Editorial spotlight on top OPS game
📈
Trends
Market intelligence dashboard (very long page)
🎮
Game Detail
Per-game autopsy (accessed via click, not in nav)

Proposed Navigation (6 pages + 2 utilities)

🔍
Discover
Blind spots, gems, creator opportunities, rising games
New
📊
Database
Full game table with enhanced filters (demo, multiplayer, subgenre)
Enhanced
🔥
Radar Pick
Editorial spotlight with shareable Radar Pick cards
Enhanced
📈
Trends
Market intelligence with TOC, section anchors, date range selector
Restructured
⭐
Watchlist
Saved games with status tags, OPS trajectory, and alerts
New
🎮
Game Detail
Per-game autopsy with section nav, share card, breakout narrative
Restructured
📤
Export (utility)
CSV/JSON export from any data view; per-section on Trends
New
🔗
Share (utility)
Share cards, permalinks, embeddable widgets
New

Persona Primary Flows → Navigation

How each persona's primary workflow maps to the proposed navigation. Highlighted pages are the new or restructured touchpoints that directly address each persona's core need.

Maya Kim — Content Creator
Daily Scouting Flow
Primary need: Find high-quality games with low YouTube coverage
Discover → Game Detail → Watchlist → Alert

Discover page eliminates the need to manually cross-reference OPS with YouTube coverage. Watchlist alerts replace daily checking.

David Park — Publisher Scout
Weekly Scouting Pipeline
Primary need: Compare game trajectories and export for CRM
Database → Game Detail → Compare → Export → Watchlist

Compare mode and CSV export feed directly into his existing CRM workflow. Watchlist tracks outreach status across weeks.

Rachel Jensen — Community Curator
Biweekly Recommendation Flow
Primary need: Find quality games and share with community
Discover → Database → Game Detail → Share → Watchlist

Share cards turn discovery into community content. Discover page surfaces gems. Watchlist serves as her recommendation history.

Alex Vasquez — Gaming Journalist
Monthly Research Flow
Primary need: Extract citable trend data and breakout narratives
Trends → Export → Game Detail → Radar Pick

Restructured Trends page with TOC and export enables his entire research workflow. Permalinks provide stable citations for articles.
