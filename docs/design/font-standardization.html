Font Rules

Four simple rules that cover every design decision.

RULE 01
All UI text is Public Sans
Headings, labels, descriptions, buttons, navigation, tooltips, filter controls. If it's not a number or the Radar hero, it's Public Sans.
RULE 02
All data values are JetBrains Mono
OPS scores, review counts, prices, percentages, deltas, chart axis labels. Use font-mono in Tailwind. Tabular figures align columns perfectly.
RULE 03
Only Radar Pick uses Playfair Display
The SignalFire hero headline is the sole consumer of Playfair Display. No other page uses a serif. This scarcity is what gives it editorial weight.
RULE 04
Never mix families within a data cluster
A stat card showing "1,247 reviews" should use mono for "1,247" and sans for "reviews" — but never sans for the number or mono for the label.
